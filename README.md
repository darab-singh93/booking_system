# Booking System API

A Node.js-based booking system with JWT authentication that allows users to book time slots between **8:00 AM to 11:59 PM IST**, with a maximum of **10 bookings per 30-minute slot**. Admins can manage bookings, and users must be authenticated to make a booking.
